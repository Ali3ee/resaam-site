#Refonte Site Web RESAAM

##Setup local 

###1. Cloner le repository

```bash
git clone Ali3ee/resaam-site
```

###2. Ouvrir le projet

Ouvrir le dossier dans VS Code.

###3. Installer les extensions recommandées

-Prettier
-Live Server
-GitLens

###4. Lancer le projet

Ouvrir `index.html`

Puis :

-clic droit
-"Open with Live Server"
